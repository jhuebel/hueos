HueOS Display Modes Feature Summary
====================================

OVERVIEW
--------
Added support for multiple text display resolutions with automatic VESA mode detection.

BOOT OPTIONS (7 total)
-----------------------
1. HueOS                                    - Standard 80x25
2. HueOS (Verbose - Detailed Hardware Info) - Standard 80x25 verbose
3. HueOS (80x50 High Resolution)            - 2x vertical lines
4. HueOS (132x25 Wide Screen)               - 1.65x horizontal columns
5. HueOS (132x43 Wide High Res)             - Wide + many lines
6. HueOS (132x50 Extra Lines)               - Maximum text space
7. HueOS (Verbose + 80x50 High Res)         - Verbose + high resolution

SUPPORTED RESOLUTIONS
---------------------
- 80x25  (Standard VGA)
- 80x50  (High vertical resolution)
- 100x37 (Balanced wide mode)
- 132x25 (Wide screen)
- 132x43 (Wide high resolution)
- 132x50 (Extra wide with many lines)

KEY FEATURES
------------
✅ Automatic screen size detection via VGA CRTC registers
✅ Command-line resolution parsing (res=WIDTHxHEIGHT)
✅ Dynamic terminal dimensions (replaces static 80x25)
✅ Fallback to 80x25 if detection fails
✅ Serial logging of detected resolution
✅ GRUB gfxpayload=text for compatibility
✅ Platform-independent (works on QEMU, Hyper-V, VirtualBox, VMware)

DETECTION ALGORITHM
-------------------
Reads VGA CRTC registers:
- 0x12: Vertical Display End (lower 8 bits)
- 0x07: Overflow register (VDE bits 8-9)
- 0x01: Horizontal Display End
- 0x09: Maximum Scan Line (character height)

Calculates:
  width  = HDE + 1
  height = (VDE + 1) / character_height

Validates: 40-200 columns, 20-100 rows

USAGE EXAMPLES
--------------
From GRUB menu:
  Select desired resolution with arrow keys + Enter

From GRUB command line (press 'e' to edit):
  multiboot /boot/hueos.bin res=132x43
  multiboot /boot/hueos.bin verbose res=80x50
